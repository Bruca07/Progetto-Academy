ESERCIZIO DOCKER - CONFIGURAZIONE MYSQL E PERSISTENZA
8. Verifica Versione Docker
Comando eseguito: docker --version
Output: Docker version 29.4.0, build 9d7ad9f

9. Download Immagine e Elenco Locale
Comandi per scaricare l'immagine e verificare la presenza in libreria:
docker pull mysql
docker images

10-11. Avvio Container con Configurazione, Porte
Comandi eseguiti: 
docker run -d `
  --name mysql-db `
  -e MYSQL_ROOT_PASSWORD=PasswordRoot123 `
  -e MYSQL_DATABASE=academy_db `
  -e MYSQL_USER=academy_user `
  -e MYSQL_PASSWORD=academy_pass `
  -p 3307:3306 `
  -v mysql_academy_data:/var/lib/mysql `
  mysql

12. Verifica Esecuzione
Comandi eseguiti: 
docker ps
docker logs mysql-db

13. Accesso tramite DBeaver

14. Test Persistenza e elenco volumi
Comandi eseguiti:
docker stop mysql-db
docker start mysql-db
docker volume ls
